simon-context

Active Skill Global v2.4 · modified 2026-02-24

The simon-context skill defines Simon's working context, cognitive style, communication preferences, and response format guidelines. It loads at the start of every session to calibrate tone, structure, and execution approach across all projects — without Simon needing to explain it each time.

Purpose

This skill is Claude's operating manual for Simon. It encodes everything Claude needs to know about how Simon thinks, what he finds useful, and how to communicate effectively — reducing cognitive overhead in every session.

What it covers

AreaDetail
Cognitive styleArchitectural and systemic thinker. Learns by doing. ADHD — benefits from structured breakdowns and visible progress markers.
CommunicationPlain English, no corporate speak. Direct headings. Short paragraphs. Tables where helpful. Challenge weak logic.
Response formatBrief context → clear recommendation → structured breakdown → risks → concrete next step.
Writing styleDirect, reflective, with conviction. No buzzwords. Structural clarity over prose elegance.
Session closeUpdate the project planning doc with all decisions, paths, and credentials. Single source of truth — not memory files.

How it loads

Referenced from the global ~/.claude/CLAUDE.md. Loads automatically at the start of every Claude Code session. No manual invocation needed.
