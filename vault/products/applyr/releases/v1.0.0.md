# Applyr v1.0.0

**Released:** 2026-03-11
**Tag:** [v1.0.0](https://github.com/sspaynter/applyr/releases/tag/v1.0.0)
**Environment:** Production at `jobs.ss-42.com`

## Summary

Initial production release of Applyr — an AI-assisted job application platform. Built across 48 commits and 66 sessions. Self-hosted on QNAP NAS, exposed via Cloudflare Tunnel.

## Changes

### Added

- Full pipeline management — job lifecycle from new through accepted/rejected/withdrawn/archived
- AI cover letter generation with voice profiles and inline feedback annotations
- Resume tailoring with inline change highlights, dual ATS scoring, and DOCX export
- AI company research briefs for interview preparation
- AI-assisted application question answers
- Role-track-based job scoring with configurable criteria (PM and IT tracks)
- n8n job ingestion — webhook endpoint, email-based discovery, automated backfill
- Review queue with batch navigation, JD highlight and annotate, inline notes
- Dashboard with home overview, pipeline kanban, metrics, notifications, and search
- Google OAuth with cross-app SSO via shared_auth schema on `.ss-42.com`
- Security hardening — rate limiting, CSP, OAuth state parameter
- 75-test automated test suite
- Docker deployment with GitHub Actions CI/CD and Watchtower auto-deploy

### Infrastructure

- Container: `applyr` on port 8084
- Database: `applyr` on `n8n-postgres`
- Image: `ghcr.io/sspaynter/applyr:latest`
- n8n workflows: 3 production copies (Job Capture Webhook, Job Alert Discovery, Job Alert Backfill)

## Rollback

DOCKER=/share/CACHEDEV2_DATA/.qpkg/container-station/usr/bin/.libs/docker
