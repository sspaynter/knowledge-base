# Knowledge Platform v2.2.0

**Released:** 2026-03-13
**Tag:** [v2.2.0](https://github.com/sspaynter/knowledge-base/releases/tag/v2.2.0)
**Previous:** v2.1.1
**Sprint:** kb-usability-1 (Navigation)

## Changes

### Added
- **Dead link indicators** — after markdown renders, `/page/` links are checked via the resolve API. Unresolvable links get red strikethrough styling with `cursor: not-allowed`. Powered by `markDeadLinks()` in `content.js`. (#59)
- **Version history viewer** — slide-in panel accessible from page header clock icon. Lists all versions with timestamps, change summaries, and author badges (vault/user/api). (#115)
- **Version content preview** — click any version row to see read-only rendered markdown content inline
- **Version restore** — one-click restore with confirmation dialog. Creates a new version entry on restore with summary "Restored version {id}"
- **GET /api/pages/:id/versions/:versionId** — new endpoint returning single version with full content body
- **7 new backend tests** — resolve endpoint (404, 400, valid slug), version list, version content, version 404, restore
