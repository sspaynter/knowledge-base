# Knowledge Platform v2.1.1

**Released:** 2026-03-05
**Tag:** [v2.1.1](https://github.com/sspaynter/knowledge-base/releases/tag/v2.1.1)
**Previous:** v2.1.0

## Changes

### Fixed
- **Sidebar page hierarchy** — `getPageTree()` now returns a flat array matching the client's recursive `renderPageTree()` pattern. Child pages are visible at the correct indented depth. Previously, the server's `buildTree()` nested children inside parent objects, making them invisible to the client's flat-array filter. (#46)
- **SESSION_SECRET alignment** — Staging and production containers now share the same `SESSION_SECRET` from `shared-secrets.env`, resolving intermittent auth overlay popups when both tabs are open. (#47)

### Removed
- `buildTree()` helper function in `services/pages.js` — dead code that conflicted with client-side tree rendering

## Files changed

| File | Change |
|---|---|
| `services/pages.js` | Removed `buildTree()` function and call; `getPageTree()` returns `res.rows` directly |
