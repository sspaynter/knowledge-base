# Decisions Register

Architecture Decision Records (ADRs) for the SS42 AI Operating Model. Each record captures a significant design choice: what was decided, what alternatives were considered, and why this option was chosen.

Decisions are permanent. If a decision is reversed, a new ADR supersedes the old one — the original stays for historical context.

## Format

Each ADR follows a standard template:

| Field | Purpose |
|---|---|
| **Decision** | What was decided |
| **Date** | When (session number) |
| **Context** | What prompted it |
| **Options considered** | What else was on the table |
| **Rationale** | Why this option won |
| **Consequences** | What this means — positive, negative, risks |
